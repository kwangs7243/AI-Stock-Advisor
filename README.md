# AI Stock Advisor Research

AI 뉴스 감성분석 기반 주식 어드바이저 연구 프로젝트입니다.

이 프로젝트는 주가를 정확히 맞히는 상용 투자 서비스를 목표로 하기보다, 약 1개월 안에 구현 가능한 MVP를 통해 뉴스 감성점수 기반 예측의 가능성과 한계를 일부 검증하고 향후 개선 방향을 제시하는 것을 목표로 합니다.
