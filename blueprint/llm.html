LLM Explanation

LLM 설명 기능

1차 프로젝트에서는 감성 흐름과 예측 참고값을 사용자가 이해하기 쉬운 문장으로 바꾸는 보조 기능입니다.

역할

예측 근거 설명 감성점수, 최근 뉴스, 가격 흐름을 묶어 설명합니다.
질의응답 1차 프로젝트에서는 복잡한 RAG보다 저장된 뉴스와 분석 결과를 바탕으로 설명을 제공합니다.
주의 문구 투자 판단을 대신하지 않고 참고 정보임을 명확히 안내합니다.

응답 생성에 사용할 데이터

  • 검색 종목의 최근 뉴스 제목과 요약
  • 뉴스별 감성 라벨과 점수
  • 기간별 종합 감성 점수
  • 주가 등락률과 거래량 변화
  • 간단한 예측 참고값과 사용된 기본 데이터
향후 확장

고도화된 RAG 시스템, 벡터DB 기반 문서 검색, 장기 대화 기록 활용은 1차 프로젝트 이후 확장 기능으로 분리합니다.
