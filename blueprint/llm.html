LLM Explanation

LLM 설명 기능

모델 결과와 뉴스 근거를 사용자가 이해하기 쉬운 문장으로 바꾸는 보조 기능입니다.

역할

예측 근거 설명 감성점수, 최근 뉴스, 가격 흐름을 묶어 설명합니다.
질의응답 사용자의 종목 관련 질문에 저장된 근거 데이터를 기반으로 응답합니다.
주의 문구 투자 판단을 대신하지 않고 참고 정보임을 명확히 안내합니다.

응답 생성에 사용할 데이터

  • 검색 종목의 최근 뉴스 제목과 요약
  • 뉴스별 감성 라벨과 점수
  • 기간별 종합 감성 점수
  • 주가 등락률과 거래량 변화
  • 회귀모델 예측 참고값과 사용된 주요 특성
