LLM 설명 기능은 최종 MVP 구현 범위에 포함하지 않습니다

현재 호출 API·프롬프트 계약·화면 연동 근거가 없으므로 구현 기능처럼 표현하지 않고 후속 검토 조건만 남깁니다.

현재 판정

호출 API구현 근거 없음
프롬프트·응답 계약확정되지 않음
Web 연동최종 MVP 제외

현재 AI 분석 카드는 DB에 저장된 Direction·RecentFlow·감성·뉴스를 조회하며, LLM이 생성한 설명이 아닙니다.

후속 도입 시 가능한 역할

  • 기존 DB 분석 결과를 사용자가 이해하기 쉬운 문장으로 재구성
  • 긍정·부정 뉴스의 근거를 함께 제시하는 요약 설명
  • 모델 출력, 데이터 기준일, 불확실성을 한 문맥에서 안내

LLM은 기존 예측 모델의 정확도를 높이는 구성요소가 아니라 결과 설명 계층으로만 검토합니다.
