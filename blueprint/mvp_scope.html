기능판정근거 또는 제한
20개 테마·100개 종목 기준 데이터 수집포함Linux 운영 로그와 DB 적재 구조 확인
멱등성 DB upsert·실행 이력포함고유키 기반 upsert와 pipeline_run 기록
뉴스 추출 요약·금융 감성 분석포함결과 저장까지 연결, 일부 레코드 실패는 별도 공개
Direction·RecentFlow 결과 저장제한적 포함기술 흐름은 구현됐으나 예측 품질은 성능 미검증
대시보드·종목 상세·검색·AI 카드포함연결 DB 환경에서 주요 읽기 요청 검증
이메일 인증·즐겨찾기제한적 포함구현 소스는 있으나 SMTP·쓰기·브라우저 E2E 미검증, 즐겨찾기는 세션 저장
LLM 설명·CI·상용 배포 기반제외구현 근거가 없어 후속 후보로만 기록

MVP 수용 기준과 결과

  1. 수집과 적재
    반복 실행 가능한 파이프라인과 DB 계약을 갖춘다. 충족
  2. AI 분석 연결
    요약·감성 결과가 Web 조회 가능한 테이블까지 이어진다. 충족
  3. 예측 결과 전달
    Direction·RecentFlow를 저장하고 화면에서 참고값으로 제공한다. 기술 충족·성능 제한
  4. 사용자 조회 경험
    주요 화면과 읽기 API가 연결 DB에서 응답한다. 충족
  5. 근거 보존
    테스트, DB 스키마, Linux 실행 로그로 결과를 재검토할 수 있다. 충족

최종 결과의 의미

이 MVP의 핵심 성과는 높은 투자 수익률을 증명하는 것이 아니라, 데이터 수집에서 AI 처리와 웹 조회까지 이어지는 서비스 구조를 짧은 기간 안에 구현한 것입니다.

“예측 모델이 동작한다”와 “예측 모델이 유효하다”는 같은 말이 아닙니다. 화면의 수치는 교육·분석 참고값이며 투자 추천으로 표현하지 않습니다.

KDT 1차 팀 프로젝트의 교육용 최종 MVP 기록입니다. 투자 판단이나 상용 운영 준비 완료를 의미하지 않습니다.
