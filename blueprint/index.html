Project Blueprint

AI 뉴스 감성분석 기반 주식 어드바이저 연구 프로젝트

주식 어드바이저 서비스를 완성품으로 만드는 데서 그치지 않고, 뉴스 감성분석과 다양한 시장 특성이 주가 예측에 어느 정도 기여하는지 실험하고 한계를 분석하는 연구형 설계 문서입니다.

프로젝트 목적

특정 종목과 관련된 뉴스를 수집하고 AI로 감성분석한 뒤, 감성점수와 주가 특성, 외부 시장 요인을 단계적으로 결합하며 예측 성능이 어떻게 변하는지 확인합니다.

핵심 문장

이 프로젝트의 핵심은 주가를 맞히겠다는 단정이 아니라, 어떤 데이터가 예측에 도움이 되고 어떤 한계가 남는지 검증하는 것입니다.

연구 방향

서비스 구현

종목 검색, 뉴스 조회, 감성 흐름, 예측 참고값을 보여주는 웹 서비스를 구현합니다.

연구와 검증

뉴스 감성만으로는 부족한 지점과 추가 데이터가 필요한 이유를 실험 결과로 설명합니다.

Research Direction 페이지에서 연구 질문과 실험 설계를 자세히 확인할 수 있습니다.

타겟 사용자

주식 초보자 뉴스와 주가 흐름을 한 화면에서 이해하고 싶은 사용자
시간이 부족한 투자자 종목 관련 뉴스 분위기를 빠르게 요약해서 보고 싶은 사용자
근거 중심 사용자 리딩방 의존을 줄이고 데이터 기반 판단을 늘리고 싶은 사용자

서비스 핵심 흐름

데이터 수집

뉴스 수집크롤링 또는 뉴스 API
주가 수집일자, 종가, 등락률, 거래량
전처리중복 제거, 날짜 정리, 종목 매핑

AI 분석

감성분석긍정, 부정, 중립 점수 생성
특성 결합감성점수와 주가 데이터 결합
회귀모델예측 참고값 생성

웹 서비스

종목 검색기업명 또는 종목명 입력
시각화뉴스 목록, 감성 흐름, 예측 결과
LLM 설명결과와 근거를 쉬운 문장으로 요약
