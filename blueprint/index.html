현재 구현 현황

영역상태현재 사실
Flask 주가표연동 완료 feature 브랜치공유 MySQL → GET /api/stockTable → 기준일 선택·정렬 표. 현재 유일한 사용자 연결 vertical slice입니다.
20개 테마 수집오프라인 구현주가·뉴스·본문 수집 코드가 CSV를 생성하지만 DB 자동 적재는 연결되지 않았습니다.
뉴스 요약·감성오프라인 구현 실행 검증 필요KoBERT 추출요약과 단일 KR-FinBERT 감성 배치·DB upsert 소스가 있습니다. 통합·운영 검증이 필요합니다.
예측실험 브랜치방향 분류와 RecentFlow 회귀 R&D입니다. 기준선 통과, DB 저장, API 서빙이 확정되지 않았습니다.
TOP3·AI 분석·검색·인증목업현재 UI 또는 고정 데이터만 존재합니다. ‘AI 추천’이 아닌 ‘AI 분석 참고값’ 시안입니다.
LLM·CI·배포계획구현 API, 자동 테스트, CI, production 배포 기반이 없습니다.

현재 연결과 단절

연결된 흐름

stock / stock_price → Flask API → 날짜 선택 → 주가 표 → 브라우저 정렬

news.clean_article_text → KoBERT 요약 → news_summary → KR-FinBERT → sentiment는 별도 배치 소스로 존재합니다.

E2E 완료 아님

수집 CSV → DB 적재, 감성·예측 → Flask API/UI 연결이 끊겨 있습니다. 예측 저장 Job과 LLM 설명 API도 없습니다.
