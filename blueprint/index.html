Project Blueprint

AI 뉴스 감성분석 기반 주식 어드바이저 연구 프로젝트

뉴스 감성분석과 프로젝트 범위 내 예측 결과를 바탕으로 뉴스 기반 주가 예측의 가능성과 한계를 분석하고, 그 결과를 사용자에게 설명할 수 있는 AI 기반 주식 어드바이저 서비스를 구현하는 설계 문서입니다.

프로젝트 목적

특정 종목과 관련된 뉴스를 수집하고 AI로 감성분석한 뒤, 감성 결과와 주가 데이터를 DB에 저장합니다. 이후 회귀모델 또는 프로젝트 범위 내 예측 결과를 웹 화면에 표시하고, LLM이 결과를 설명하도록 구성합니다.

핵심 문장

이 프로젝트의 핵심은 정확한 예측 자체가 아니라 데이터 수집, Feature Engineering, 모델 학습, 결과 분석, 한계 분석, 향후 개선 방향 제시까지 포함하는 전체 개발 과정을 구현하는 것입니다.

프로젝트 메시지

본 프로젝트는 AI를 이용해 주가를 완벽하게 예측하는 서비스를 만드는 것이 아니라, 뉴스 감성분석과 다양한 데이터 특성이 주가 예측에 어떤 영향을 주는지 실험하고, 그 결과와 한계를 사용자에게 설명할 수 있는 AI 기반 주식 어드바이저 서비스를 구현하는 것을 목표로 합니다.

1차 프로젝트 구현 범위

필수 구현

뉴스 수집, 전처리, 감성분석, DB 저장, 주가 데이터 저장, 회귀모델 학습, Flask 화면, LLM 설명을 구현합니다.

결과 분석

예측값을 정답처럼 제공하지 않고, 실제 결과와 비교해 모델의 한계와 개선 방향을 정리합니다.

연구 방향 페이지에서 연구 질문과 실험 설계를 자세히 확인할 수 있습니다.

타겟 사용자

주식 초보자 뉴스와 주가 흐름을 한 화면에서 이해하고 싶은 사용자
시간이 부족한 투자자 종목 관련 뉴스 분위기를 빠르게 요약해서 보고 싶은 사용자
근거 중심 사용자 리딩방 의존을 줄이고 데이터 기반 판단을 늘리고 싶은 사용자

서비스 핵심 흐름

데이터 수집

뉴스 수집크롤링 또는 뉴스 API
주가 수집일자, 종가, 등락률, 거래량
전처리중복 제거, 날짜 정리, 종목 매핑

AI 분석

감성분석긍정, 부정, 중립 점수 생성
특성 결합감성점수와 주가 데이터 결합
예측 기능회귀모델 또는 프로젝트 범위 내 예측값 생성

웹 서비스

종목 검색기업명 또는 종목명 입력
시각화뉴스 목록, 감성 흐름, 예측 결과
LLM 설명결과와 근거를 쉬운 문장으로 요약

향후 연구 및 확장

자동 재학습, 고도화된 RAG, 벡터DB, 관리자 페이지, 다단계 Baseline 비교 등은 1차 프로젝트 이후의 확장 가능 기능으로 분리합니다.
