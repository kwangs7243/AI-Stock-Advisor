Team project overview

흩어진 주식 정보와 뉴스를 한 화면에서 이해하도록 돕는 서비스

PILOS는 주가와 뉴스를 자동으로 모으고, AI로 핵심 내용과 분위기를 분석해 웹 화면으로 보여주는 교육용 주식 분석 서비스입니다.

3주 2일 팀 프로젝트 · 2026-07-16 기준 · 상세 기술 근거는 각 문서에서 확인할 수 있습니다.

어떤 문제를 해결하려 했나요?

정보가 흩어져 있습니다주가, 관련 뉴스, 시장 분위기를 여러 곳에서 따로 찾아야 합니다.
뉴스가 너무 많습니다많은 기사를 모두 읽고 중요한 내용을 찾는 데 시간이 걸립니다.
한눈에 보기 어렵습니다숫자와 뉴스가 서로 떨어져 있어 종목의 최근 흐름을 이해하기 어렵습니다.

우리 팀의 해결 방법

주가와 뉴스를 자동으로 모음
        ↓
긴 뉴스에서 핵심 문장을 추리고 긍정·부정 흐름을 분석
        ↓
분석 결과를 데이터베이스에 저장
        ↓
웹 대시보드와 종목 상세 화면에서 한 번에 확인

발표에서는 모델명이나 테이블명보다 “수집 → 분석 → 저장 → 화면 제공”의 흐름을 먼저 설명하면 됩니다.

종목 상세주가 차트, 뉴스 분위기, AI 분석 참고값을 함께 봅니다.구현
종목 검색회사명이나 종목코드로 원하는 종목을 찾습니다.구현
뉴스 요약·감성긴 뉴스의 핵심 내용과 긍정·부정 흐름을 빠르게 확인합니다.구현
방향·흐름 분석모델이 계산한 다음 흐름을 참고 정보로 확인합니다.성능 검증 필요

발표에서 바로 사용할 소개 문장

예측 결과는 참고값입니다

모델이 실행되고 결과가 저장되는 과정은 구현했지만, 높은 예측 정확도나 수익성을 증명한 것은 아닙니다.

모든 기능을 상용 수준으로 검증한 것은 아닙니다

주요 조회 화면은 확인했지만 이메일 전송, 데이터 쓰기, 실제 브라우저 전체 흐름은 검증 범위가 제한됐습니다.

더 자세히 보고 싶다면

서비스가 연결되는 구조수집부터 웹 화면까지의 전체 흐름
실제로 동작한 근거테스트, 데이터베이스 점검, 서버 실행 로그
모델 결과와 한계예측 성능을 과장하지 않는 평가 내용

교육용 프로젝트이며 투자 판단이나 수익을 보장하는 서비스가 아닙니다.
