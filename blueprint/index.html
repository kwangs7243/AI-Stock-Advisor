Project Blueprint

AI 뉴스 감성분석 기반 주식 어드바이저 서비스

주식 초보자와 바쁜 사용자가 자극적인 정보에 휘둘리지 않고, 뉴스 흐름과 감성점수, 예측 근거를 함께 확인하며 더 주도적으로 판단할 수 있도록 돕는 서비스 설계 문서입니다.

프로젝트 목적

특정 종목과 관련된 뉴스를 수집하고 AI로 감성분석하여, 사용자가 최근 뉴스 분위기와 투자 판단에 참고할 수 있는 정보를 쉽게 확인하도록 돕습니다.

핵심 문장

감성분석은 최종 목적이 아니라, 주가 판단을 돕는 데이터 특성을 만드는 과정입니다.

타겟 사용자

주식 초보자 뉴스와 주가 흐름을 한 화면에서 이해하고 싶은 사용자
시간이 부족한 투자자 종목 관련 뉴스 분위기를 빠르게 요약해서 보고 싶은 사용자
근거 중심 사용자 리딩방 의존을 줄이고 데이터 기반 판단을 늘리고 싶은 사용자

서비스 핵심 흐름

데이터 수집

뉴스 수집크롤링 또는 뉴스 API
주가 수집일자, 종가, 등락률, 거래량
전처리중복 제거, 날짜 정리, 종목 매핑

AI 분석

감성분석긍정, 부정, 중립 점수 생성
특성 결합감성점수와 주가 데이터 결합
회귀모델예측 참고값 생성

웹 서비스

종목 검색기업명 또는 종목명 입력
시각화뉴스 목록, 감성 흐름, 예측 결과
LLM 설명결과와 근거를 쉬운 문장으로 요약
