Project Overview

AI 뉴스 감성분석 기반 주식 어드바이저

뉴스 감성분석과 주가 데이터를 결합해 예측 가능성과 한계를 확인하고, 그 결과를 사용자가 이해할 수 있도록 설명하는 연구형 서비스 프로젝트입니다.

프로젝트 목적

본 프로젝트는 AI로 주가를 완벽하게 예측하는 것이 아니라, 뉴스 감성분석과 데이터 Feature가 주가 예측에 어떤 영향을 주는지 실험하고 그 결과와 한계를 설명하는 서비스를 구현하는 것을 목표로 합니다.

프로젝트 목표

데이터 흐름 구현뉴스 수집부터 Flask 출력까지 연결합니다.
Feature 설계감성점수, 주가, 뉴스 개수, 키워드 정보를 모델 입력으로 구성합니다.
결과 설명LLM은 예측 모델이 아니라 분석 결과를 설명하는 역할로 제한합니다.

핵심 아이디어

뉴스를 데이터로 바꾸기

수집게임 카테고리 뉴스부터 시작
요약원문보다 요약본을 감성분석 입력으로 사용
감성점수회귀모델 Feature로 활용

주가 예측 실험

최근 3일 Window입력 데이터 구조 고정
단일 예측 목표MVP에서는 하나의 y만 선택
3개 모델 비교모델 성능보다 흐름 검증 우선

서비스로 확인

DB 공유뉴스, 감성, 예측 결과를 중앙 저장
Flask 출력데이터 전달 여부를 확인
LLM 설명결과와 한계를 쉬운 문장으로 설명

개발 철학

원칙의미
정확한 예측보다 구조 검증모델 성능보다 데이터가 끝까지 흐르는지 먼저 확인합니다.
감성분석은 Feature 생성 과정감성분석 자체가 목적이 아니라 회귀모델 입력을 만들기 위한 단계입니다.
모델은 분리해서 관리카테고리별 감성모델, 예측 목표별 회귀모델을 분리합니다.
확장 가능한 구조새 카테고리, 새 Feature, 새 모델을 기존 구조에 덧붙일 수 있게 설계합니다.

문서 읽는 순서

1. 전체 아키텍처서비스 계층과 데이터 흐름을 확인합니다.
2. MVP 구현 계획이번 단계에서 구현할 범위와 완료 조건을 확인합니다.
3. 모델 설계뉴스 처리, Feature, 감성모델, 회귀모델 구조를 확인합니다.
