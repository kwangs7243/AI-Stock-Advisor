단계별 계획

단계작업산출물
1단계뉴스 수집, 전처리, 주가 데이터 저장 구조 구현수집 스크립트, DB 스키마
2단계감성분석 적용과 감성 결과 DB 저장뉴스별 감성분석 결과
3단계감성점수와 주가 데이터를 결합한 예측 기능 구현회귀모델 또는 프로젝트 범위 내 예측 결과
4단계Flask 화면 구현뉴스 목록, 감성 흐름, 예측 참고값 화면
5단계LLM 설명 기능 추가결과 요약과 한계 설명
6단계예측 결과 분석과 발표자료 정리실패 원인 분석, 향후 개선 방향

1차 프로젝트 이후 확장

자동 재학습, 관리자 페이지, 고도화된 RAG, 벡터DB, Prediction Feedback API, 다단계 Baseline 비교는 기간 내 필수 구현이 아니라 후속 연구 및 서비스 고도화 후보로 둡니다.
