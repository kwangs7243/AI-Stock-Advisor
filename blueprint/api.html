Flask API

API 설계 초안

프론트 화면에서 Flask 서버로 요청할 주요 API 후보입니다.

엔드포인트 후보

MethodPath역할응답 후보
GET/api/stocks/search종목 검색종목 코드, 기업명, 시장
GET/api/stocks/{stock_id}/news관련 뉴스 조회뉴스 목록, 감성 라벨, 점수
GET/api/stocks/{stock_id}/sentiment기간별 감성 흐름날짜별 긍정/부정 비율, 종합 점수
GET/api/stocks/{stock_id}/prediction예측 참고 정보예측값, 기준일, 모델 버전
POST/api/llm/explainAI 설명 생성요약 설명, 참고 뉴스, 주의 문구

화면 구성 후보

  • 검색 입력 영역
  • 뉴스 목록과 감성 라벨
  • 기간별 감성 흐름 그래프
  • 예측 참고값과 주요 근거
  • LLM 설명 영역
