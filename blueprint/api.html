현재 Flask 라우트와 연결 DB 스모크 검증 결과를 기준으로 조회 기능과 검증 제한을 구분합니다.

발표에서는 이렇게 설명하세요

“API는 데이터베이스에 저장된 정보를 웹 화면으로 전달하는 통로입니다. 사용자는 대시보드, 종목 상세, 검색, AI 분석 카드에서 그 결과를 확인합니다.”

발표 본문에서는 API 주소를 모두 읽기보다 사용자가 볼 수 있는 기능을 중심으로 설명합니다.

날짜 계약

analysisDate

주가표·감성처럼 분석 기준일이 필요한 조회에 사용하며, 값이 없으면 저장된 최신 기준일을 선택합니다.

priceDate

AI 카드가 특정 가격 기준일에 연결된 Direction·RecentFlow와 뉴스를 조회할 때 사용합니다.

스모크 검증 범위

/, /api/stockTable, /api/themeCharts, /api/search?q=삼성, /stock/000660, /api/chart_data/000660, /api/sentiment_chart/000660, /api/ai_card/000660의 8개 요청이 연결 DB 환경에서 모두 HTTP 200을 반환했습니다.

이 결과는 읽기 응답의 스모크 검증입니다. 실제 브라우저 상호작용, SMTP 전송, 회원 데이터 쓰기, 세션 지속성까지 통과했다는 뜻은 아닙니다. 즐겨찾기는 현재 user_stock에 영속 저장하지 않고 세션을 사용합니다.

KDT 1차 팀 프로젝트의 교육용 최종 MVP 기록입니다. 투자 판단이나 상용 운영 준비 완료를 의미하지 않습니다.
