Flask API

API 설계 초안

MVP 화면 구현에 필요한 최소 API 후보입니다. 자동 피드백 저장 API는 현재 구현 범위에서 제외합니다.

엔드포인트 후보

MethodPath역할응답 후보
GET/api/stocks/search종목 검색종목 코드, 기업명, 시장
GET/api/stocks/{stock_id}/news관련 뉴스 조회뉴스 목록, 감성 라벨, 점수
GET/api/stocks/{stock_id}/sentiment기간별 감성 흐름날짜별 긍정/부정 비율, 종합 점수
GET/api/stocks/{stock_id}/prediction간단한 예측 참고 정보예측값, 기준일, 모델 버전
POST/api/llm/explainAI 설명 생성요약 설명, 참고 뉴스, 주의 문구

화면 구성 후보

  • 검색 입력 영역
  • 뉴스 목록과 감성 라벨
  • 기간별 감성 흐름 그래프
  • 예측 참고값과 주요 근거
  • LLM 설명 영역
Prediction Feedback 처리

현재 단계에서는 자동 API나 Ajax 기반 저장 기능으로 구현하지 않습니다. 발표와 결과 분석에서는 예측 결과와 실제 결과를 비교해 모델의 한계를 설명하고, 향후 개선 방향을 제시하는 연구 개념으로 다룹니다.
