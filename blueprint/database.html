그룹테이블역할
기준·사용자stock, model, user종목 기준정보, 모델 메타데이터, 사용자 정보
운영pipeline_run파이프라인 모드·상태·시작·종료·메시지
원천 데이터news, stock_price뉴스 본문과 일자별 주가
텍스트 분석news_summary, summary_analysis, summary_analysis_map요약, 감성 분석, 원문·요약·분석 연결
예측features, direction, recentflow모델 입력 피처, 방향 분류, 단기 흐름 점수

핵심 관계

stock ─┬─ stock_price
       ├─ news ─ news_summary ─ summary_analysis_map ─ summary_analysis
       ├─ features ─ direction
       ├─ features ─ recentflow
       └─ user_stock ─ user

model ─ direction
model ─ recentflow
pipeline_run ─ news_summary / summary_analysis / features / direction
pipeline_run ··· recentflow.pipeline_run_id  (논리 참조)

recentflow.pipeline_run_id는 운영 이력을 가리키는 논리 연결이지만 현재 DDL에는 물리 FK가 없습니다. ERD에서도 점선 관계로 해석해야 합니다.

최신화 기준

  • 정식 스키마 파일: pilos-ai/Db/schema.sql
  • 운영 DB 대조 스냅샷: pilos-ai/docs/db_schema_snapshot.json
  • ERD 문서: pilos-ai/docs/erd.md

prediction·sentiment는 현재 운영 스키마의 테이블이 아닙니다. 감성은 summary_analysis, 예측은 direction과 recentflow에 저장됩니다.

KDT 1차 팀 프로젝트의 교육용 최종 MVP 기록입니다. 투자 판단이나 상용 운영 준비 완료를 의미하지 않습니다.
