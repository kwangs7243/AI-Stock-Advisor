ML Pipeline

모델 파이프라인

1차 프로젝트에서는 뉴스 감성분석, 기본 특성 생성, 회귀모델 학습, 결과 분석 흐름을 구현합니다.

처리 단계

1. 수집 및 전처리

뉴스 수집종목 키워드 기준 뉴스 수집
텍스트 정제HTML 제거, 중복 제거, 결측치 처리
날짜 정리뉴스 날짜와 주가 날짜 매칭 준비

2. 감성분석

모델 후보BERT, KoBERT, scikit-learn 모델
결과긍정, 부정, 중립 라벨과 점수
저장뉴스별 감성 결과를 DB에 저장

3. 예측 및 분석

기본 특성 생성감성점수와 최소한의 주가 정보 결합
회귀모델 학습회귀모델 또는 프로젝트 범위 내 예측값 생성
결과 분석예측값과 실제 결과의 차이 해석

1차 프로젝트 저장 파일 후보

파일설명
sentiment_model.pkl 또는 .h5뉴스 감성분석 모델
regression_model.pkl간단한 주가 예측 참고값 생성 모델
tokenizer텍스트 토큰화에 사용하는 자원
scaler.pkl필요한 경우에만 사용하는 기본 스케일링 도구

향후 연구 방향

다양한 Feature Engineering, 여러 Baseline 모델 비교, 시장지수·수급·환율·금리 등 외부 데이터 결합은 1차 프로젝트 이후 성능 개선을 위해 검토할 확장 연구로 분리합니다.
