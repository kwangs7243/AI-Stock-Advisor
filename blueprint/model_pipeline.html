ML Pipeline

모델 파이프라인

뉴스 텍스트를 감성점수로 바꾸고, 주가 데이터와 결합해 예측 참고 정보를 생성하는 흐름입니다.

처리 단계

1. 수집 및 전처리

뉴스 수집종목 키워드 기준 뉴스 수집
텍스트 정제HTML 제거, 중복 제거, 결측치 처리
날짜 정리뉴스 날짜와 주가 날짜 매칭 준비

2. 감성분석

모델 후보BERT, KoBERT, scikit-learn 모델
결과긍정, 부정, 중립 라벨과 점수
저장뉴스별 감성 결과를 DB에 저장

3. 예측 모델

특성 생성감성점수, 등락률, 거래량, 이동평균
모델 후보LinearRegression, RandomForest, XGBoost
결과 제공예측값과 사용 특성을 저장

저장 파일 후보

파일설명
sentiment_model.pkl 또는 .h5뉴스 감성분석 모델
regression_model.pkl주가 예측 참고값 생성 모델
tokenizer텍스트 토큰화에 사용하는 자원
scaler.pkl수치형 특성 스케일링 도구
