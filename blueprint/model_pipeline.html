알려진 위험

영역위험필요한 계약
요약코드 최대 1,000자와 DB varchar(500) 불일치, 실패 재시도 차단 가능성길이·상태·attempt·error·immutable revision
감성사용한 요약 revision이 보존되지 않고 다중 결과가 덮일 수 있음news_summary_id 또는 summary lineage
예측단순 기준선을 넘지 못한 저장 실험이 있으며 persistence/API 없음시간 검증, baseline gate, artifact·feature schema 버전

Production 승격 기준

  1. walk-forward 또는 rolling validation과 leakage 방지
  2. 고정 평가셋에서 단순 기준선 초과
  3. 기간·종목별 안정성과 감성 feature lift 확인
  4. model artifact, feature schema, data snapshot, revision 고정
  5. prediction 저장과 실제값 사후 비교·drift 추적
  6. API/UI에 모델 버전·기준일·불확실성·비권유 문구 제공

카테고리별 감성 모델과 production registry는 계획입니다.
