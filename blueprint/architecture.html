발표에서는 이렇게 설명하세요

“데이터를 모으는 AI 서버와 결과를 보여주는 웹 서버가 같은 데이터 저장소를 사용합니다. AI가 분석 결과를 저장하면 웹이 그 결과를 읽어 사용자 화면에 보여줍니다.”

쉽게 말하면: 이 페이지는 서비스의 부품들이 어떤 순서로 연결되는지 보여주는 상세 근거입니다.

운영 경계와 외부 의존성

실행 제어

pipeline_run이 모드, 상태, 시작·종료 시각과 실행 메시지를 기록합니다. 중복 데이터는 upsert 계약으로 다룹니다.

외부 경계

데이터 제공 사이트, Hugging Face 모델 파일, MySQL, SMTP는 외부 상태에 영향을 받습니다. Web은 AI 프로세스를 직접 실행하지 않습니다.

KDT 1차 팀 프로젝트의 교육용 최종 MVP 기록입니다. 투자 판단이나 상용 운영 준비 완료를 의미하지 않습니다.
